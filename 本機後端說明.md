# 教務輔導決策平台：本機後端

## 啟動方式

在 Finder 開啟「教務輔導資料」，雙擊 `啟動本機網站.command`，再使用瀏覽器開啟：

`http://127.0.0.1:8765/`

終端機視窗必須保持開啟；關閉視窗後，本機網站會停止。

## 第一階段已完成

- 前台上傳檔案會實際寫入分校資料夾。
- 新分校會建立固定的 `00` 至 `05`、`90`、`99` 分類。
- 高信心檔名會先放入對應分類；無法判斷的檔案放入 `99_待分類`。
- 每批上傳會在 `90_分析產出/上傳批次` 建立 JSON 清冊。
- 上傳後會自動解析 Excel、Word、文字型 PDF 與圖片 OCR。
- 解析結果會寫入 `90_分析產出/內容解析`，並顯示在分校儀表板。
- 分校、上傳與指派狀態保存在 `00_資料索引/教務輔導.db`。
- 新分校會同步更新 `00_資料索引/分校清冊.csv`。
- 單批上傳容量上限為 500MB。

## 尚未啟用或仍有限制

- 錄音轉錄
- 掃描型 PDF 逐頁 OCR
- 完整公司規範數值比對與進階綜合分析

狀態「部分完成」表示部分檔案已解析，但仍有資料夾外殼、錄音、掃描型 PDF 或不支援格式等待處理。
